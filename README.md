# 7zip cookbook


nstall 7zip

## Supported Platforms

Windows

## Attributes

default['tool']['url'] = "http://downloads.sourceforge.net/sevenzip/7z920-x64.msi"
default['tool']['file'] = "C:/Program Files/7-zip/7z.exe"

## Usage

Add 7zip:default to your run list to install the 7zip application.
